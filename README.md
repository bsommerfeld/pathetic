# archaeology 🥀⛏️

You found the dig site.

This is the FibonacciHeap Pathetic used before we learned better — preserved exactly as it
came from [jheaps](https://github.com/d-michail/jheaps) by Dimitrios Michail (Apache-2.0,
original header intact, `LICENSE` included — museum rules).

It allocated objects like a crypto miner on Christmas. It pointer-chased like it was getting
paid per cache miss. It was, the literature assures us, "amortized O(1)".

The benchmarks disagreed. It lost to a binary heap. The binary heap later lost to a
quaternary heap on dense node ids. Evolution is brutal like that.

Nothing in this branch compiles. Nothing in this branch is supposed to. Pay your respects
and return to [`trunk`](../../tree/trunk), where the living heaps are.
